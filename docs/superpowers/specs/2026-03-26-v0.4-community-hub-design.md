# v0.4 Community Hub — Design Spec

## Overview

为 A2A-Forge 添加社区功能：Agent 目录、共享测试集、健康监控、收藏夹。让用户从"单机使用"进化到"社区协作"。

## 数据模型

### 新增 SQLite 表

```sql
-- 社区 Agent 目录（本地缓存）
CREATE TABLE community_agents (
    id           TEXT PRIMARY KEY,
    name         TEXT NOT NULL,
    description  TEXT DEFAULT '',
    url          TEXT NOT NULL UNIQUE,
    card_json    TEXT NOT NULL,
    tags         TEXT DEFAULT '[]',       -- JSON array of tags
    author       TEXT DEFAULT '',
    stars        INTEGER DEFAULT 0,
    last_checked TEXT DEFAULT (datetime('now')),
    created_at   TEXT DEFAULT (datetime('now'))
);

-- 收藏夹
CREATE TABLE favorites (
    id        TEXT PRIMARY KEY,
    agent_id  TEXT NOT NULL,
    folder    TEXT DEFAULT 'default',     -- folder/category name
    notes     TEXT DEFAULT '',
    created_at TEXT DEFAULT (datetime('now')),
    FOREIGN KEY (agent_id) REFERENCES agents(id) ON DELETE CASCADE
);

-- Agent 健康检查记录
CREATE TABLE health_checks (
    id         TEXT PRIMARY KEY,
    agent_id   TEXT NOT NULL,
    status     TEXT NOT NULL,             -- 'healthy' | 'degraded' | 'down'
    latency_ms INTEGER,
    error      TEXT,
    checked_at TEXT DEFAULT (datetime('now')),
    FOREIGN KEY (agent_id) REFERENCES agents(id) ON DELETE CASCADE
);
```

## Rust 后端

### 新增 Commands (`commands/community.rs`)

| Command | 说明 |
|---------|------|
| `list_community_agents(search?, tags?)` | 搜索社区 Agent |
| `add_from_community(url, workspace_id)` | 从社区添加 Agent |
| `submit_to_community(agent_id)` | 提交 Agent 到社区目录 |
| `toggle_favorite(agent_id, folder?)` | 收藏/取消收藏 |
| `list_favorites(folder?)` | 列出收藏 |
| `update_favorite(id, folder?, notes?)` | 更新收藏信息 |
| `check_agent_health(agent_id)` | 检查单个 Agent 健康 |
| `check_all_health(workspace_id)` | 检查所有 Agent 健康 |
| `list_health_checks(agent_id, limit?)` | 查看健康历史 |
| `export_test_suite(suite_id)` | 导出测试套件为 JSON |
| `import_test_suite(json_data, workspace_id)` | 导入测试套件 |

## 验收标准

1. 可以浏览和搜索社区 Agent 目录
2. 可以一键从社区添加 Agent 到工作空间
3. 收藏夹支持文件夹分组和备注
4. 健康监控可以检查 Agent 可用性和延迟
5. 测试套件可以导出为 JSON 并导入
