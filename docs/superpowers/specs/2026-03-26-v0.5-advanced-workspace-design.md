# v0.5 Advanced Workspace — Design Spec

## Overview

增强工作空间能力：导出/导入完整配置、环境变量替换、请求链（技能输出串联）、Diff 视图（跨运行对比）。

## 数据模型

### 新增 SQLite 表

```sql
-- 环境变量
CREATE TABLE env_variables (
    id           TEXT PRIMARY KEY,
    workspace_id TEXT NOT NULL,
    name         TEXT NOT NULL,
    value        TEXT NOT NULL,
    is_secret    INTEGER DEFAULT 0,
    created_at   TEXT DEFAULT (datetime('now')),
    FOREIGN KEY (workspace_id) REFERENCES workspaces(id) ON DELETE CASCADE,
    UNIQUE(workspace_id, name)
);

-- 请求链
CREATE TABLE request_chains (
    id           TEXT PRIMARY KEY,
    name         TEXT NOT NULL,
    description  TEXT DEFAULT '',
    workspace_id TEXT NOT NULL,
    created_at   TEXT DEFAULT (datetime('now')),
    FOREIGN KEY (workspace_id) REFERENCES workspaces(id) ON DELETE CASCADE
);

-- 请求链步骤
CREATE TABLE chain_steps (
    id            TEXT PRIMARY KEY,
    chain_id      TEXT NOT NULL,
    sort_order    INTEGER NOT NULL,
    name          TEXT NOT NULL,
    agent_id      TEXT NOT NULL,
    skill_name    TEXT NOT NULL,
    request_json  TEXT NOT NULL,
    extract_json  TEXT DEFAULT '{}',   -- JSONPath mappings: {"varName": "$.result.url"}
    timeout_ms    INTEGER DEFAULT 60000,
    FOREIGN KEY (chain_id) REFERENCES request_chains(id) ON DELETE CASCADE,
    FOREIGN KEY (agent_id) REFERENCES agents(id) ON DELETE CASCADE
);
```

## Rust 后端 Commands

| Command | 说明 |
|---------|------|
| `export_workspace(workspace_id)` | 导出完整工作空间为 JSON |
| `import_workspace(json_data)` | 导入工作空间 |
| `list_env_vars(workspace_id)` | 列出环境变量 |
| `set_env_var(workspace_id, name, value, is_secret)` | 设置变量 |
| `delete_env_var(id)` | 删除变量 |
| `create_chain(name, desc, workspace_id)` | 创建请求链 |
| `update_chain(id, name?, desc?)` | 更新链 |
| `delete_chain(id)` | 删除链 |
| `list_chains(workspace_id)` | 列出链 |
| `add_chain_step(...)` | 添加步骤 |
| `update_chain_step(...)` | 更新步骤 |
| `delete_chain_step(id)` | 删除步骤 |
| `run_chain(chain_id)` | 执行链 |
| `diff_responses(response_a, response_b)` | 对比两个响应 |
