Features
Everything you need to test A2A agents
From discovery to debugging — a complete toolkit for the A2A protocol workflow.
🔍

Agent Discovery

Fetch and inspect .well-known/agent.json agent cards. Browse skills, capabilities, and metadata instantly.

⚡

Skill Testing

Invoke any skill with adaptive input — text, JSON, or file upload. Auto-polls async tasks until completion.

📡

Live Streaming

SSE-based streaming with real-time status updates. Watch agent responses arrive token by token.

🖼️

Smart Preview

Auto-detect and render images, video, and audio inline. Deep-scans JSON results for media URLs.

🧪

Test Suites

Group test cases into automated sequences with 7 assertion types. Export results as HTML or JSON.

🔐

Secure Auth

Per-agent default headers with OS keychain credential storage. Credentials never leave your machine.

🌐

Local Registry Proxy

Embedded proxy for agent discovery, request interception, latency simulation, and traffic replay.

🔗

Request Chaining

Pipe output of one skill into another with JSONPath extraction. Build multi-step agent workflows.

👥

Community Hub

Discover popular A2A agents, share test collections, and monitor agent health with periodic checks.

Quick Start
Up and running in 60 seconds
# Clone and run
git clone https://github.com/learningpro/a2a-forge.git
cd a2a-forge
npm install
npm run tauri dev

# Build for production
npm run tauri build

Requires Rust (stable) and Node.js v18+. See Tauri prerequisites for platform-specific deps.
