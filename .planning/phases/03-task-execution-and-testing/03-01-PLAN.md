    All commands use `#[tauri::command]` + `#[specta::specta]` attributes. All use `crate::db::get_pool(&app)` for DB access.

    1. **commands/tasks.rs**: Three commands:
       - `send_task(agent_url: String, payload: serde_json::Value, auth_header: Option<String>, extra_headers: Option<HashMap<String, String>>, timeout_secs: Option<u64>, state: tauri::State<'_, AppState>, app: tauri::AppHandle) -> Result<serde_json::Value, AppError>` — Calls `client::send_task_rpc`, using timeout_secs (default 30). Records start time, returns response JSON.
       - `stream_task(agent_url: String, payload: serde_json::Value, auth_header: Option<String>, extra_headers: Option<HashMap<String, String>>, on_event: Channel<TaskEvent>, state: tauri::State<'_, AppState>) -> Result<String, AppError>` — Creates EventSource from `client::build_sse_request`, spawns tokio task to iterate events via `futures::StreamExt::next()`, sends each parsed `TaskEvent` through `on_event.send()`. Registers `AbortHandle` in `state.active_tasks`. Returns task_id (UUID).
       - `cancel_task(task_id: String, state: tauri::State<'_, AppState>) -> Result<(), AppError>` — Looks up task_id in `state.active_tasks`, calls `.abort()`, removes entry. Returns Ok(()) even if not found.

    2. **commands/history.rs**: Three commands:
       - `save_history(agent_id: String, skill_name: String, request_json: String, response_json: Option<String>, status: String, duration_ms: Option<i64>, app: tauri::AppHandle) -> Result<String, AppError>` — INSERT into history table with UUID id, current unix timestamp. Returns the id.
       - `list_history(agent_id: Option<String>, search: Option<String>, limit: Option<i64>, app: tauri::AppHandle) -> Result<serde_json::Value, AppError>` — SELECT from history, optionally filtered by agent_id and search (LIKE on skill_name). Order by created_at DESC. Limit default 100. Returns JSON array of row objects.
       - `clear_history(agent_id: Option<String>, app: tauri::AppHandle) -> Result<(), AppError>` — DELETE from history WHERE agent_id = ? (if provided) or DELETE all.

    3. **commands/saved_tests.rs**: Three commands:
       - `save_test(name: String, agent_id: String, skill_name: String, request_json: String, app: tauri::AppHandle) -> Result<String, AppError>` — INSERT into saved_tests with UUID id, current unix timestamp. Returns id.
       - `list_saved_tests(agent_id: Option<String>, skill_name: Option<String>, app: tauri::AppHandle) -> Result<serde_json::Value, AppError>` — SELECT from saved_tests, optionally filtered. Returns JSON array.
       - `delete_saved_test(test_id: String, app: tauri::AppHandle) -> Result<(), AppError>` — DELETE from saved_tests WHERE id = ?.

    4. **commands/workspaces.rs**: Four commands:
       - `list_workspaces(app: tauri::AppHandle) -> Result<serde_json::Value, AppError>` — SELECT all from workspaces ORDER BY created_at. Returns JSON array.

       - `delete_workspace(workspace_id: String, app: tauri::AppHandle) -> Result<(), AppError>` — Prevent deleting "default" workspace (return AppError::Database). DELETE otherwise.
       - `set_active_workspace(workspace_id: String, app: tauri::AppHandle) -> Result<(), AppError>` — Verify workspace exists (SELECT), return NotFound if not. This is a frontend-only concern but validates existence.
