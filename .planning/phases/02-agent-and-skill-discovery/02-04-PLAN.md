3. Export handler:
```typescript
async function handleExport() {
  try {
    const { save } = await import("@tauri-apps/plugin-dialog");
    const filePath = await save({
      defaultPath: "agents-export.json",
      filters: [{ name: "JSON", extensions: ["json"] }],
    });
    if (!filePath) return; // user cancelled
    const jsonData = await commands.exportAgents("default");
    // Write the file — use tauri-plugin-fs or a Rust helper
    // Since Plan 01's export_agents returns a JSON string, we need to write it to the file
    // Use the writeTextFile from @tauri-apps/plugin-fs if available, or
    // use the Rust command to write directly
    // Simplest: use @tauri-apps/api for file writing
    const { writeTextFile } = await import("@tauri-apps/plugin-fs");
    await writeTextFile(filePath, jsonData);
  } catch (e) {
    console.error("Export failed:", e);
  }
}
```

NOTE: If `@tauri-apps/plugin-fs` is not installed, install it (`npm install @tauri-apps/plugin-fs`, add `tauri-plugin-fs` to Cargo.toml, register plugin, add permissions). OR: modify the Rust `export_agents` command to accept a file_path parameter and write the file in Rust using `tokio::fs::write`. The Rust approach is cleaner — avoids adding another plugin dependency.

Alternative cleaner approach: Modify the Rust `export_agents` and `import_agents` commands in `agents.rs` to handle file I/O directly:
- `export_agents(workspace_id, file_path)` — query DB, serialize, write to file_path via `tokio::fs::write`
- `import_agents(file_path, workspace_id)` — read file via `tokio::fs::read_to_string`, parse, fetch each card, save

If modifying Rust commands, also update lib.rs command registration and regenerate bindings.

If not modifying Rust: Use `@tauri-apps/plugin-fs` with `writeTextFile` and `readTextFile`. Add `"fs:allow-read"` and `"fs:allow-write"` to capabilities.

Choose the approach that works — prefer avoiding new plugin dependencies.

4. Import handler:
```typescript
async function handleImport() {
  try {
    const { open } = await import("@tauri-apps/plugin-dialog");
    const filePath = await open({
      filters: [{ name: "JSON", extensions: ["json"] }],
      multiple: false,
    });
    if (!filePath || Array.isArray(filePath)) return;
    // Read the file and call importAgents
    // Then reload agents
    const { readTextFile } = await import("@tauri-apps/plugin-fs");
    const jsonData = await readTextFile(filePath as string);
    await commands.importAgents(jsonData, "default");
    await useAgentStore.getState().loadAgents("default");
  } catch (e) {
    console.error("Import failed:", e);
  }
}
```
