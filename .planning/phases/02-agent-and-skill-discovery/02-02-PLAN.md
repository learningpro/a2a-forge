Replace the placeholder agentStore with a full CRUD store. Import `commands` and types from `../bindings` (the exact import path depends on what tauri-specta generated — read bindings.ts first).

The store interface:
```typescript
interface AgentState {
  agents: AgentRow[];
  selectedAgentId: string | null;
  selectedSkillId: string | null;
  isLoading: boolean;
  error: string | null;

  loadAgents: (workspaceId: string) => Promise<void>;
  addAgent: (baseUrl: string, nickname: string | null, workspaceId: string) => Promise<AgentRow>;
  deleteAgent: (agentId: string) => Promise<void>;
  refreshAgent: (agentId: string) => Promise<void>;
  setSelectedAgentId: (id: string | null) => void;
  setSelectedSkillId: (id: string | null) => void;
  selectedAgent: () => AgentRow | undefined;
}
```

Implementation details:
- `loadAgents`: set isLoading=true, call `commands.listAgents(workspaceId)`, set agents array, set isLoading=false. On error, set error string.
- `addAgent`: call `commands.addAgent(baseUrl, nickname, workspaceId)`, append result to agents array, return the new AgentRow.
- `deleteAgent`: call `commands.deleteAgent(agentId)`, remove from agents array. If selectedAgentId === agentId, set selectedAgentId to null.
- `refreshAgent`: call `commands.refreshAgent(agentId)`, replace the agent in the array with the updated one.
- `setSelectedAgentId`: set selectedAgentId, also reset selectedSkillId to null.
- `setSelectedSkillId`: set selectedSkillId.
- `selectedAgent`: return `agents.find(a => a.id === selectedAgentId)`.

Do NOT use persist middleware on agentStore — agents are persisted in SQLite, not localStorage.
